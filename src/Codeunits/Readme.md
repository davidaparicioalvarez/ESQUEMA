# Codeunits
English: Containers for business logic (procedures, event subscribers, calculations).
Spanish: Contenedores de lógica de negocio (procedimientos, suscriptores de eventos, cálculos).
Guidelines:
- Single responsibility.
- PascalCase object names, camelCase variables.
- Group related procedures; keep each <50 lines.
- Add XML docs and ESP= comments inside AL files.
